■ サービス概要
　減量（体重を落とす）の管理が上手くできない悩みを持った人に
　適切な管理ができて減量後の楽しみを与える
　減量管理サービスです

■　メインのターゲットユーザー
　17〜30代前半の男性、プロボクサー、ボディビルダー、ダイエットしたい人

■ ユーザーが抱える課題
　体重を期間までに計画的に落としていきたいが、紙に書いて管理するのもデータを作成するのも大変。
　また食べ物、飲み物が欲しいけど、ほとんど飲まず食べすなので減量後の楽しみが欲しい。

■ 解決方法
　アプリを使う事によって手軽に計画的に体重を落としていくことができる。

■ 実装予定の機能
　　　・目標日まであと何日か分かるカウントダウン
　　　・現在の体重と基礎代謝を残りの日数を入力すると、オススメの計画表が作成される機能
　　　・自分で体重を落とす計画を設定する事もできる機能
　　　・減量後に食べたい物をお気に入り登録できる機能
　　　・日毎にランダムでやる気アップの名言が出る機能
　　　

■　なぜこのサービスを作りたいのか？
　自分が実際に減量を行なっていた時に計画と体重管理に困っていた為。
